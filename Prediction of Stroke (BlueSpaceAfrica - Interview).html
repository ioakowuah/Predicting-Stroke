#Imputting missing values
class DataFrameImputer(TransformerMixin):

    def __init__(self):
        """Impute missing values.

        Columns of dtype object are imputed with the most frequent value 
        in column.

        Columns of other types are imputed with mean of column.

        """
    def fit(self, X, y=None):

        self.fill = pd.Series([X[c].value_counts().index[0]
            if X[c].dtype == np.dtype('O') else X[c].median() for c in X],
            index=X.columns)

        return self

    def transform(self, X, y=None):
        return X.fillna(self.fill)


joined_data = DataFrameImputer().fit_transform(joined_data)

Conclusion¶

Here the optimal model has ensemble four classifiers which is Decision Tree Classifier, Random Forest Classifier, Logistics Regression and XGBoost. This model has 98% predictive accuracy. Except for the Logistics Regression Classifier, all the other three classifiers identify age, bmi and avg_glucose_level to be the leader feature of prediction.
